utils.filter_op.size_filter(bboxes, threshs)

Remove all bboxes with any side smaller than min_size or larger than max_size.

Parameters:

  • bboxes (np array N x 4) – Input bboxes.
  • threshs (list/tuple 2 [min, max]) – Min and Max size threshs of bboxes

Returns:

  • bboxes (np array M x 4) – BBoxes after filtering size.

utils.filter_op.ratio_filter(bboxes, threshs)

Remove too fat or too tall bboxes.

Parameters:

  • bboxes (np array N x 4) – Input bboxes.
  • threshs (list/tuple 2 [min, max]) – Min and Max aspect ratio threshs of bboxes.

Returns:

  • bboxes (np array M x 4) – BBoxes after filtering aspect ratio.

utils.filter_op.border_filter(bboxes, threshs)

Remove all bboxes out side image.

Parameters:

  • bboxes (np array N x 4) – Input bboxes.
  • threshs (list/tuple 2 [width, heigth]) – Width and height of image.

Returns:

  • bboxes (np array M x 4) – BBoxes after removing bboxes out side image.

utils.filter_op.hard_fg_filter(bboxes, hard_fgs, thresh)

Remove all bboxes having a large overlap with hard foreground.

Parameters:

  • bboxes (np array N x 4) – Input bboxes.
  • hard_fgs (np array A x 4) – Hard foreground.
  • thresh (float) – Iou thresh.

Returns:

  • bboxes (np array M x 4) – BBoxes after removing bboxes having a large overlap with hard foreground.

utils.filter_op.dont_care_filter(bboxes, dont_care_areas, thresh)

Remove all bboxes having a large overlap with don’t care area.

Parameters:

  • bboxes (np array N x 4) – Input bboxes.
  • dont_care_area (np array A x 4) – don’t care area.
  • thresh (float) – Iou thresh.

Returns:

  • bboxes (np array M x 4) – BBoxes after removing bboxes having a large overlap with don’t care area.

utils.filter_op.nms_filter(bboxes, scores, thresh, is_sorted=False)

Do non-maximum suppression to bboxes.

Parameters:

  • bboxes (np array N x 4) – Input bboxes.
  • scores (list/tuple/np array N) – scores of input bboxes.
  • thresh (float) – Iou thresh.
  • is_sorted (bool) – If true the input bboxes and scores is sorted.

Returns:

  • keep_ids (list M) – BBoxes after no-maximum suppression.

utils.select_fg_bg.select_fg_bg(bboxes, gt_bboxes, gt_cat_ids, fg_rate, bboxes_per_img, fg_threshs, bg_threshs)

Select foregrounds and backgrounds from bboxes.

Parameters:

  • gt_bboxes (np array G x 4) – Ground truth bboxes.
  • gt_cat_ids (np array G) – Category ids of ground truth.
  • fg_rate (float) – The fg rate chosen for training.
  • bboxes_per_img (int) – The bbox number chosen for training.

  • bg_threshs (list) – The low and high thresh of bg

  • cat_ids (np array M) – Category ids of selected bboxes.
  • reg_targets (np array M x 4) – Deltas of bboxes.
  • reg_weights (np array M x 4) – Weights of bboxes.
  • fg_num (int) – Number of foreground.
